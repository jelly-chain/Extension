# polygon-skill

Polygon.io options chains, unusual flow detection, IV surface, historical OHLCV, news

## Keys required
POLYGON_API_KEY

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the polygon-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
