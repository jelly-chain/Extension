# sec-edgar-skill

SEC EDGAR filing retrieval, full-text search, XBRL structured data, 10-K/10-Q/8-K parsing

## Keys required
SEC_USER_AGENT (email string)

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the sec-edgar-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
