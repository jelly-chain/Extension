# alpaca-skill

Alpaca stock & ETF paper/live trading — orders, positions, market data, WebSocket streaming

## Keys required
ALPACA_API_KEY, ALPACA_API_SECRET

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the alpaca-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
