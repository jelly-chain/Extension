# alpha-vantage-skill

Alpha Vantage fundamentals, earnings, income statements, balance sheets, DCF, RSI/MACD

## Keys required
ALPHA_VANTAGE_API_KEY

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the alpha-vantage-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
