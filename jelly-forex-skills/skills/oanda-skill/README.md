# oanda-skill

OANDA forex & CFD practice/live trading — 70+ pairs, pip calculator, candles, streaming prices

## Keys required
OANDA_API_TOKEN, OANDA_ACCOUNT_ID, OANDA_PRACTICE=true

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the oanda-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
