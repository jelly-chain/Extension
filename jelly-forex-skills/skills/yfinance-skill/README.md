# yfinance-skill

yfinance Python library — historical OHLCV, dividends, earnings calendar, screeners (no key needed)

## Keys required
None — public Yahoo Finance data

## Install

**Mac/Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

## Usage
After installing, reference this skill in Claude Code:
```
Use the yfinance-skill to ...
```

See [SKILL.md](./SKILL.md) for full API examples.
