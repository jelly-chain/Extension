# web-scraper

Scrape, crawl, and extract structured data from any website.

## Required skills
firecrawl-skill, exa-skill, serpapi-skill

## Required keys
FIRECRAWL_API_KEY, EXA_API_KEY, SERPAPI_KEY

## Install
```bash
cp agent.md ~/.claude/agents/web-scraper.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent web-scraper
```
