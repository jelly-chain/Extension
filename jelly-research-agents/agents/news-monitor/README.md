# news-monitor

Real-time news tracking and daily briefings across thousands of sources.

## Required skills
newsapi-skill, perplexity-skill, tavily-skill

## Required keys
NEWSAPI_KEY, PERPLEXITY_API_KEY, TAVILY_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/news-monitor.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent news-monitor
```
