# competitive-intel

Map competitive landscapes, build competitor profiles, and track company news.

## Required skills
perplexity-skill, exa-skill, firecrawl-skill, serpapi-skill, newsapi-skill

## Required keys
PERPLEXITY_API_KEY, EXA_API_KEY, FIRECRAWL_API_KEY, SERPAPI_KEY, NEWSAPI_KEY

## Install
```bash
cp agent.md ~/.claude/agents/competitive-intel.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent competitive-intel
```
