# deep-researcher

Multi-step research chains with Perplexity + Exa + Firecrawl producing comprehensive cited reports.

## Required skills
perplexity-skill, exa-skill, tavily-skill, firecrawl-skill

## Required keys
PERPLEXITY_API_KEY, EXA_API_KEY, TAVILY_API_KEY, FIRECRAWL_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/deep-researcher.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent deep-researcher
```
