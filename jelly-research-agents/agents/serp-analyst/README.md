# serp-analyst

Analyse Google SERP results, extract SERP features, and profile ranking pages.

## Required skills
serpapi-skill, firecrawl-skill, exa-skill

## Required keys
SERPAPI_KEY, FIRECRAWL_API_KEY, EXA_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/serp-analyst.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent serp-analyst
```
