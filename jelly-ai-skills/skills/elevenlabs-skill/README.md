# elevenlabs-skill

Text-to-speech, voice cloning, speech-to-speech, and sound effects via ElevenLabs API.

## Install
```bash
bash install.sh
```

## Required key
`ELEVENLABS_API_KEY — get one at elevenlabs.io`

## Key location
`~/.jelly-ai/.keys`
