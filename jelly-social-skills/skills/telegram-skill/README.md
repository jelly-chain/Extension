# telegram-skill

Telegram Bot API (node-telegram-bot-api) — send messages, images, polls, inline keyboards to users and groups.

## Install
```bash
bash install.sh
```

## Setup required
1. Open [@BotFather](https://t.me/BotFather) in Telegram → /newbot → follow prompts
2. Copy your Bot Token
3. Get your Chat ID from [@userinfobot](https://t.me/userinfobot) or `/getUpdates`
4. Add keys to `~/.jelly-social/.keys`

## Example prompts
- "Send a Telegram message to my channel: 'Alert: market opened'"
- "Broadcast an update to these 3 Telegram groups"
- "Set up a bot that responds to /price with the current SOL price"
- "Send a poll asking which token to trade next"
- "Schedule a daily 9am Telegram briefing"
