# newsapi-skill

Real-time news headlines and article search from 80,000+ sources worldwide.

## Install
```bash
bash install.sh
```

## Setup required
1. Get your API key (see below)
2. Add key to `~/.jelly-research/.keys`

## Required key
`NEWSAPI_KEY — get one at newsapi.org`
