# serpapi-skill

Google, Bing, News, Scholar and more — returns clean structured JSON from any SERP.

## Install
```bash
bash install.sh
```

## Setup required
1. Get your API key (see below)
2. Add key to `~/.jelly-research/.keys`

## Required key
`SERPAPI_KEY — get one at serpapi.com`
