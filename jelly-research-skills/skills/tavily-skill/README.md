LLM-optimised search with built-in synthesis, Q&A endpoint, and content extraction.

## Install
```bash
bash install.sh
```

## Setup required
1. Get your API key (see below)
2. Add key to `~/.jelly-research/.keys`

`TAVILY_API_KEY — get one at app.tavily.com`
