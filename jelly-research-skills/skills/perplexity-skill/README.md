# perplexity-skill

AI-powered search that synthesises cited answers from live web data.

## Install
```bash
bash install.sh
```

## Setup required
1. Get your API key (see below)
2. Add key to `~/.jelly-research/.keys`

## Required key
`PERPLEXITY_API_KEY — get one at perplexity.ai`
