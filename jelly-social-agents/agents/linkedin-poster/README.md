# linkedin-poster

Post professional updates and share articles on LinkedIn.

## Required skills
linkedin-skill

## Required keys
LINKEDIN_CLIENT_ID, LINKEDIN_CLIENT_SECRET, LINKEDIN_ACCESS_TOKEN

## Install
```bash
cp agent.md ~/.claude/agents/linkedin-poster.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent linkedin-poster
```
