# discord-moderator

Send messages, manage roles, and moderate Discord servers.

## Required skills
discord-skill

## Required keys
DISCORD_BOT_TOKEN, DISCORD_WEBHOOK_URL, DISCORD_APP_ID, DISCORD_GUILD_ID

## Install
```bash
cp agent.md ~/.claude/agents/discord-moderator.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent discord-moderator
```
