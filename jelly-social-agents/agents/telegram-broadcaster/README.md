# telegram-broadcaster

Send messages, broadcasts, and alerts to Telegram users and groups.

## Required skills
telegram-skill

## Required keys
TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID

## Install
```bash
cp agent.md ~/.claude/agents/telegram-broadcaster.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent telegram-broadcaster
```
