# tweet-scheduler

Post tweets, threads, and media on X via the Twitter API v2.

## Required skills
twitter-skill

## Required keys
TWITTER_API_KEY, TWITTER_API_SECRET, TWITTER_BEARER_TOKEN, TWITTER_ACCESS_TOKEN, TWITTER_ACCESS_SECRET

## Install
```bash
cp agent.md ~/.claude/agents/tweet-scheduler.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent tweet-scheduler
```
