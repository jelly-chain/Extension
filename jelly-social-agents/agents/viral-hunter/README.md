# viral-hunter

Discover viral content and emerging trends across Twitter, Reddit, and TikTok.

## Required skills
twitter-skill, reddit-skill, tiktok-skill

## Required keys
TWITTER_BEARER_TOKEN, REDDIT_CLIENT_ID, TIKTOK_ACCESS_TOKEN

## Install
```bash
cp agent.md ~/.claude/agents/viral-hunter.md
```

Or install all agents at once:
```bash
bash ../../install-all.sh
```

## Usage
Inside Claude Code:
```
/agent viral-hunter
```
