# voice-cloner

Text-to-speech, voice cloning (IVC/PVC), and batch audio production via ElevenLabs.

## Required skills
elevenlabs-skill

## Required keys
ELEVENLABS_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/voice-cloner.md
```
Or install all: `bash ../../install-all.sh`

## Usage
Inside Claude Code: `/agent voice-cloner`
