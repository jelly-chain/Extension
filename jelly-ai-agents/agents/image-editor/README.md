# image-editor

Remove backgrounds, replace objects, outpaint, upscale, and style-transfer images.

## Required skills
stability-skill, replicate-skill

## Required keys
STABILITY_API_KEY, REPLICATE_API_TOKEN

## Install
```bash
cp agent.md ~/.claude/agents/image-editor.md
```
Or install all: `bash ../../install-all.sh`

## Usage
Inside Claude Code: `/agent image-editor`
