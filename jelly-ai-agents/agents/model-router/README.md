# model-router

Select the right AI model for any task — routing strategies, cost optimisation, fallback chains.

## Required skills
openrouter-skill, openai-skill, groq-skill

## Required keys
OPENROUTER_API_KEY, OPENAI_API_KEY (optional), GROQ_API_KEY (optional)

## Install
```bash
cp agent.md ~/.claude/agents/model-router.md
```
Or install all: `bash ../../install-all.sh`

## Usage
Inside Claude Code: `/agent model-router`
