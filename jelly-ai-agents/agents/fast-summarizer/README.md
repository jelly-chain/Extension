# fast-summarizer

Bulk text processing at 800+ t/s — summaries, extraction, classification, meeting notes.

## Required skills
groq-skill

## Required keys
GROQ_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/fast-summarizer.md
```
Or install all: `bash ../../install-all.sh`

## Usage
Inside Claude Code: `/agent fast-summarizer`
