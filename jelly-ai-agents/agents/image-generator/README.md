# image-generator

Create images from text with Flux, SDXL, and SD3 — with prompt enhancement and iteration.

## Required skills
replicate-skill, stability-skill

## Required keys
REPLICATE_API_TOKEN, STABILITY_API_KEY

## Install
```bash
cp agent.md ~/.claude/agents/image-generator.md
```
Or install all: `bash ../../install-all.sh`

## Usage
Inside Claude Code: `/agent image-generator`
