# Jelly-AI Agents

> 6 ready-to-use AI agent templates for the Jelly-AI multi-tool agent.

**GitHub:** [github.com/jelly-chain/Extension/jelly-ai-agents](https://github.com/jelly-chain/Extension/jelly-ai-agents)

---

## Install all agents

```bash
bash install-all.sh       # Mac / Linux
.\install-all.ps1         # Windows PowerShell
```

## Use an agent

Inside Claude Code:
```
/agent gpt-coder
/agent image-generator
/agent model-router
```

---

## Agent list (6 agents)

| Agent | Description |
|-------|-------------|
| `gpt-coder` | Production code generation with GPT-4o — TypeScript, Python, tests, code review |
| `fast-summarizer` | Bulk text processing at 800+ t/s — summaries, extraction, classification |
| `image-generator` | Create images from text with Flux, SDXL, and SD3 — prompts and iteration |
| `voice-cloner` | Text-to-speech, IVC/PVC voice cloning, and batch audio production |
| `image-editor` | Remove backgrounds, inpaint, outpaint, upscale, and style-transfer images |
| `model-router` | Select and route to the right AI model — cost optimisation and fallback chains |
