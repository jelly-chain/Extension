# forex-trader

Forex trading on OANDA — practice mode by default, pip calculator, SMA crossover signals, stop-loss enforced

## Required skills
oanda-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/forex-trader.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\forex-trader.md"
```

## Usage
Inside Claude Code:
```
/agent forex-trader
```

See [agent.md](./agent.md) for full system prompt and example prompts.
