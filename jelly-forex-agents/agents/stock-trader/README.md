# stock-trader

Commission-free stock & ETF trading on Alpaca — paper trading by default, bracket orders, position sizing

## Required skills
alpaca-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/stock-trader.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\stock-trader.md"
```

## Usage
Inside Claude Code:
```
/agent stock-trader
```

See [agent.md](./agent.md) for full system prompt and example prompts.
