# portfolio-tracker

Multi-account portfolio aggregator — Alpaca stocks + OANDA forex, P&L, sector allocation, concentration alerts

## Required skills
alpaca-skill, oanda-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/portfolio-tracker.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\portfolio-tracker.md"
```

## Usage
Inside Claude Code:
```
/agent portfolio-tracker
```

See [agent.md](./agent.md) for full system prompt and example prompts.
