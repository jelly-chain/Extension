# filing-reader

SEC EDGAR filing researcher — 10-K/10-Q/8-K retrieval, XBRL data, full-text search, red flag detection

## Required skills
sec-edgar-skill, alpha-vantage-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/filing-reader.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\filing-reader.md"
```

## Usage
Inside Claude Code:
```
/agent filing-reader
```

See [agent.md](./agent.md) for full system prompt and example prompts.
