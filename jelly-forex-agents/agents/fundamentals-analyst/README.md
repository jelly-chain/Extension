# fundamentals-analyst

Full fundamental analysis — earnings, DCF valuation, margin trends, peer comparison via Alpha Vantage + yfinance

## Required skills
alpha-vantage-skill, yfinance-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/fundamentals-analyst.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\fundamentals-analyst.md"
```

## Usage
Inside Claude Code:
```
/agent fundamentals-analyst
```

See [agent.md](./agent.md) for full system prompt and example prompts.
