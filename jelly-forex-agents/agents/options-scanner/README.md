# options-scanner

Unusual options flow detection via Polygon.io — vol/OI screening, IV surface, break-even analysis

## Required skills
polygon-skill, alpha-vantage-skill

## Install

**Mac/Linux:**
```bash
cp agent.md ~/.claude/agents/options-scanner.md
```

**Windows (PowerShell):**
```powershell
Copy-Item agent.md "$env:USERPROFILE\.claude\agents\options-scanner.md"
```

## Usage
Inside Claude Code:
```
/agent options-scanner
```

See [agent.md](./agent.md) for full system prompt and example prompts.
